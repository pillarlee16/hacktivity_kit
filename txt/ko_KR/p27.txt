Superheroes of the Open Web Assessment

Badge NAVIGATOR
Type Skill
Learning Goals
• Understand the location and use of the address bar in the browser
• 탭 브라우징 기능을 이해한다
• 컴퓨터의 "복사", "붙여넣기" 기능을 마스터한다
• "HTML"이란 용어를 이해한다
• 웹 브라우징의 기초 네비게이션 스킬을 획득한다
• 마우스를 사용하여 웹 페이지의 물체들을 가리켜보고, 클릭해보고 스크롤해본다
HACKtivities to Complete
• 웹 브라우저를 열어본다
• 브라우저에 주소 창이 어디있는지 확인한다
• 주소 창에 웹 사이트 주소를 입력해본다
• 여러개의 브라우저 탭을 열어보고 그것들을 번갈아가며 사용해본다
• "복사", "붙여넣기" 기능을 마스터한다
• 마우스를 사용하여 페이지를 스크롤해본다


Badge REMIXER
Type Skill
Learning Goals
• 콘텐츠를 감싸고 있고 의미를 부여하는 요소 태그를 확인하고 수정해본다
• 요소 속성들의 내용을 재조합해다 (속성은 여는 태그 안에 나타나며 항상 따옴표로 감싸진다)
• 웹 페이지를 구성하는 코드의 블록이 되는 요소를 재조합하고 만들어본다
• HTML과 <p> </p> 태그를 사용하여 단락을 작성할 수 있는 능력을 보여준다
• 로컬 해킹과 클라우드 해킹의 차이를 이해한다
• 웹 상에 개발자들이 이용할 수 있는 자원들이 있다는 것을 이해한다
• Mozilla가 "hack" & "webmaking" 이라는 말에 대해 의미하는 바를 이해한다
HACKtivities to Complete
• 웹 사이트에 간다
• 고글을 사용하여 사이트에 있는 이미지를 교체한다
• 고글을 사용하여 웹 페이지에 글자를 변경하거나 추가한다
• 동료들과 hacked page를 공유한다
• hackbook/resource 페이지를 보기 위해 hackasaurus.org 를 다시 방문한다

Type Specialist
